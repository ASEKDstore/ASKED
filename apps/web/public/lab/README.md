# LAB Mascot

Поместите файл изображения маскота сюда:

**Имя файла:** `mascot.png`

**Путь:** `apps/web/public/lab/mascot.png`

**Рекомендации:**
- Формат: PNG с прозрачным фоном
- Размер: 240x240px или больше (квадратное изображение)
- Фон: прозрачный (PNG с альфа-каналом)

После добавления файла маскот автоматически появится на главной странице с анимацией левитации.
